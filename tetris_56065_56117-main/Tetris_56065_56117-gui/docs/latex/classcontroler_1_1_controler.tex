Lance le jeu Tetris. 

Cette méthode contrôle le déroulement du jeu Tetris en bouclant jusqu\textquotesingle{}à ce que le jeu se termine ou que le joueur gagne.

Le jeu est affiché à l\textquotesingle{}écran, puis une boucle est exécutée tant que le jeu n\textquotesingle{}est pas terminé et que le joueur n\textquotesingle{}a pas gagné. À chaque itération de la boucle, la méthode Handle\+Input() est appelée pour gérer l\textquotesingle{}entrée utilisateur, puis la méthode \doxylink{classcontroler_1_1_controler_a75057e4d52de48126524bf06a2b0929f}{update\+Game()} est appelée pour mettre à jour l\textquotesingle{}état du jeu. Enfin, le jeu mis à jour est à nouveau affiché à l\textquotesingle{}écran. \Hypertarget{classcontroler_1_1_controler_a75057e4d52de48126524bf06a2b0929f}\label{classcontroler_1_1_controler_a75057e4d52de48126524bf06a2b0929f}
